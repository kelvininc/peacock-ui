	acc[sp] = styleSpacing[sp].match(regex)[1];
	return acc;

# Spatial System
*Space is everywhere. The organization of space is key to every great design, providing **consistent rhythm** to all our products.*

Our spatial system uses a 4-point grid system for sizing, spacing, and laying out components relative to one another. This means that anything is always a multiple of 4 pixels: a margin or padding, a button height, a space between input forms, etc.  
This makes it easy to compose all our components neatly and consistently, with no space for inconsistency.


<div class="spacing-container">
	{
		Object.keys(spacings).map(sp => (
			<div key={sp} class="spacing-wrapper">
				<div class="box-wrapper">
					<div class="box" style={{width: spacings[sp], height: spacings[sp], }}></div>
				</div>
				<div class="description">
					<span class="code">${sp}</span>
					<span class="value">{spacings[sp]}</span>
				</div>
			</div>
		))
	}
</div>

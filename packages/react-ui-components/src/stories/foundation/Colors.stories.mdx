# Colors
Color is a ** functional ** and ** emotional ** tool. Our color system communicates brand personality, sets a tone, attracts attention, and indicates importance. In other words, it allows the creation of a web interface more usable and helps us consistent experiences across products.  

## Main Colors

Every main color can have these * variations *
*(variations can only be used to complement the base color)*:

* __base__: main color
* __contrast__: color that will provide readable text against the base color
* __dark__: variation that should be a darker version of the base color(used in the * hover * state of * Button *, for example) 
* __light__: variation that should be a lighter version of the base color(used in the * disable * state of * Button *, for example)

## Neutral Colors
The *Neutral Colors* consisted colors that **inform, create visual balance and harmony**. These colors are used to provide variations in text and background colors of components.  
**Note**: Should be *Background Color* stepping towards *Text Color*.

** Note **: All colors can be replaced by setting the respective * CSS Variable * <br></br>
> *** Example ***: Setting the * Primary Color *
```
--kv-primary: #005cc7;
--kv-primary-rgb: 0, 92, 199;
--kv-primary-contrast: #fff;
--kv-primary-contrast-rgb: 255, 255, 255;
--kv-primary-dark: #0051af;
--kv-primary-light: #ccdef4;
```
